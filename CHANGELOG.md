# Changelog

All notable changes to the Kult: Divinity Lost GM Screen module will be documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] — Initial Release

### Added
- GM Moves panel: all 9 categories (Regular, Elysium, Madness, Passion, Dream, Underworld, Metropolis, Inferno, Gaia)
- Rules panel: Keep it Together, Avoid Harm, Endure Injury, Stability track, Relation Moves
- Harm panel: wound penalties, typical harm, weapon harm, wound descriptions
- Unique Moves panel: Combat, Influence, Magic — all tiers 1–5, filterable by category
- NPC Builder: tag-cloud selector for Body, Face, Eyes, Clothes, Drive; random NPC generator; Send to Chat button
- Horror Contract panel: all 9 GM principles
- Experience panel: checkable session questions with session reset
- GM Chat panel: in-module command system with quick-action buttons
- `/kult` Foundry chat commands: open, roll, move, npc, harm, wound, stability, help
- Global search bar filtering all GM Moves
- Click any move chip to whisper it to GM-only Foundry chat
- `Alt + G` keyboard shortcut to toggle the screen
- Sidebar button in Foundry chat panel
- Module settings: default tab, screen position, remember last tab
- Foundry v11/v12 compatibility
